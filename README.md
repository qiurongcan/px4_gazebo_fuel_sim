# px4_gazebo_fuel_sim

## 仿真环境
|名称|具体版本|
|:----|:----|
|操作系统|Ubuntu20.04|
|ROS|ROS1 Noetic|
|Gazebo|Gazebo11|
|PX4|v1.14.0(这个版本不影响，配置上有略微区别）|
|传感器|Mid360|
|定位算法|Fast LIO2|
|搜索算法|FUEL|
